📖 About
This repository contains my solutions for 100 Java programming practice tasks designed to strengthen both fundamental and advanced Java concepts.

The objective is not only to complete the exercises, but to deeply understand the concepts behind them, document my learning process, and continuously improve my code quality through refactoring and best practices.

This repository serves as a personal knowledge base and a record of my evolution as a Java developer.

🎯 Goals
Through these challenges, I aim to:

Master Java fundamentals
Strengthen Object-Oriented Programming skills
Learn common Data Structures and Algorithms
Understand Java Collections Framework
Work with Files, Networking, and Multithreading
Practice Design Patterns and Software Design principles
Improve problem-solving abilities
Build habits of writing clean, maintainable code
Prepare for backend development with Spring Boot and related technologies
